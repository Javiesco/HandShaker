HANDSHAKER

Vista Index:

Se muestra una barra de buscador en el que podremos escribir el servicio que necesitamos encontrar dependiendo de los servicios que existen en la base de datos. También se podran buscar servicios por categorías. Solo se mostraran los servicios que hay disponibles y los usuarios que son capaces de desempeñarlos, sin poder acceder al perfil de esos usuarios. Para poder acceder a esos perfiles habra que registrarse en nuestra página y pinchar en el nombre del servicio

Vista Registro:

En la ventana de registro tendremos que rellenar los campos solicitados. Se dan unos servicios predeterminados para añadir las habilidades y los intereses de un usuario. Se usara la API de google maps para mostrar nuestra localización

Vista Administrador:

(IMPORTANTE: REGISTRARSE como administrador admin -> admin) En la ventana del administrador podremos controlar distintos factores Elimiar usuario, modificar usuario, añadir servicio, eliminar servicio, eliminar negociación, etc. según una tabla para de Usuarios,Servicios y Negociaciones

Vista Mi Historial:

Se ira rellenando automáticamente a medida que el usuario vaya desempeñando servicios. Se muestra una tabla con los trabajos realizados y ofrecidos, la fecha en la que se ha realizado ese trabajo y la valoración de cada uno de ellos.

Vista Mi Perfil:

(NECESITAS ESTAR REGISTRADO) Se mostrarán los datos del usuario, tanto los intereses como las actividades que es capaz de desempeñar. También aparecerá una tabla con la disponibilidad de horarios. Se le da al usuario la capacidad de poder añadir un nuevo usuario a la base de datos, y posteriormente añadir eses nuevo servicio como habilidad o interes También aparecerá la valoración general del usuario. Se usará la API de google maps para mostrar la localización del usuario.

Vista Mis Ofertas:

Ventana que muestra los servicios que han sido ofrecidos y enviados a ese usuario. Se podrá aceptar o rechazar las ofertas ofrecidas y tambien se podrá negociar o cancelar las enviadas.

Vista Negociación:

Ventana que muestra la conversación privada con un usuario preciso con el fin de llegar a un acuerdo de servicios. Sera capaz de cancelar esa negociación si el usuario no esta conforme, tambien se podra aceptar esa oferta si se ha llegado a un acuerdo.

Vista Servicio:

(NECESITAS ESTAR RESGISTRADO Y LOGUEADO) Ventana que aparece cuando seleccionas un servicio. Se muestran los usuarios que realizan ese servicio, pudiendole dar la opcion de intercambiar un servicio con él o no, y esto para mas de un usuario.A cada usuario le puedes ofrecer un trabajo distinto o bien dejar la negociación para después.
